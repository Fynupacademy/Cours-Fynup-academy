Guide d'utilisation
Astuce : Vos données sont sauvegardées automatiquement dans le cloud. Vous pouvez accéder à votre budget depuis n'importe quel appareil.
1. Premiers pas
1
Créer votre compte — Cliquez sur « Créer un compte » sur la page de connexion. Entrez votre email et un mot de passe (minimum 6 caractères). Votre compte est créé instantanément.
2
Choisir votre devise — En haut à droite, cliquez sur CHF ou EUR selon votre pays. Vous pouvez changer à tout moment.
3
Mode sombre / clair — Cliquez sur l'icône lune ou soleil en haut à droite pour changer l'apparence. Votre préférence est sauvegardée.
2. Saisir une transaction
1
Onglet Transactions — Cliquez sur l'onglet "Transactions" dans la barre de navigation.
2
Remplir le formulaire — Sélectionnez la date, le type (Revenus / Dépenses / Frais fixes / Crédits / Épargne), la catégorie et le montant.
3
Description optionnelle — Vous pouvez ajouter une note (ex : Migros, Salaire mars...) pour retrouver vos transactions facilement.
4
Enregistrer — Cliquez sur « + Enregistrer ». La transaction apparaît immédiatement dans l'historique ci-dessous.
5
Supprimer — Cliquez sur le bouton "x" à droite d'une transaction pour la supprimer.
3. Budget prévisionnel
1
Onglet Budget prévisionnel — Sélectionnez le mois en haut, puis entrez vos montants prévus par catégorie.
2
Sauvegarder — Cliquez sur « Sauvegarder » après avoir rempli vos montants. Ils seront visibles dans le tableau de bord.
3
Budget vs Réel — Dans le tableau de bord, une barre de progression compare ce que vous avez prévu à ce que vous avez réellement dépensé. Vert = sous le budget, orange = proche, rouge = dépassement.
4. Objectifs épargne
1
Créer un objectif — Cliquez sur « + Nouvel objectif », donnez un nom (ex : Vacances, Voiture), un montant cible et le montant déjà épargné.
2
Alimenter un objectif — Chaque mois, entrez le montant que vous ajoutez dans le champ « + Ajouter un montant » et cliquez sur « + Ajouter ». La barre de progression se met à jour.
3
Supprimer un objectif — Cliquez sur le "x" en haut à droite de la carte objectif.
5. Tableau de bord et graphiques
1
Choisir le mois — Utilisez le sélecteur de mois en haut pour naviguer entre les mois.
2
KPIs — Les 5 cartes affichent vos Revenus, Dépenses totales, Frais fixes, Épargne et Solde restant du mois sélectionné.
3
Graphique dépenses — Le donut montre la répartition de vos dépenses par catégorie pour le mois sélectionné.
4
Graphique 6 mois — Les barres comparent vos revenus et dépenses sur les 6 derniers mois pour voir votre évolution.
6. Vue annuelle et export
1
Vue annuelle — L'onglet « Vue annuelle » affiche un graphique et un résumé de chaque mois de l'année sélectionnée.
2
Export Excel — Dans le tableau de bord, cliquez sur "Export Excel" pour télécharger toutes vos transactions du mois en fichier .xlsx.
